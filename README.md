# 🍗 Zaid Chicken Centre

**Premium Quality Fresh Chicken with Home Delivery**

A modern, animated e-commerce website for Zaid Chicken Centre - a real business serving fresh, hygienic, and halal chicken since 2015.

---

## 🌐 Website Features

- **5 Fully Responsive Pages** - Home, Menu, About, Contact, Order
- **Shopping Cart System** - Add/remove items with localStorage persistence
- **WhatsApp Integration** - One-click order placement
- **Modern Dark Theme** - Glassmorphism UI with animations
- **Product Catalog** - 15+ products across 4 categories
- **Home Delivery** - Complete checkout flow with delivery form

---

## 📸 Pages Overview

| Page | Description |
|------|-------------|
| **Home** | Hero section, stats, featured products, testimonials |
| **Menu** | Product grid with search & category filters |
| **About** | Company story, mission, timeline, founder |
| **Contact** | Contact form, map, business info |
| **Order** | Cart summary, delivery form, WhatsApp checkout |

---

## 🛠️ Tech Stack

- **HTML5** - Semantic structure
- **CSS3** - Custom properties, animations, glassmorphism
- **Vanilla JavaScript** - No frameworks, pure JS
- **LocalStorage** - Cart persistence

---

## 🚀 Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Or use a local server for best experience
